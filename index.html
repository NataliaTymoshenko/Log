🔄 Пригадаймо базові функції

Перед тим як вивчати логіку, освіжимо пам'ять

Σ

SUM

=SUM(A1:A10)
Додає всі числа в діапазоні. Наприклад, підрахунок загальної суми покупок.
x̄

AVERAGE

=AVERAGE(B1:B10)
Обчислює середнє арифметичне. Корисно для обчислення середнього балу учня.
↓

MIN

=MIN(C1:C20)
Знаходить найменше значення. Наприклад, найнижчу ціну товару.
↑

MAX

=MAX(D1:D15)
Знаходить найбільше значення. Наприклад, максимальну температуру за місяць.

💡 Пам'ятаєш абсолютні посилання?

$A$1 — абсолютне посилання (не змінюється при копіюванні)
A1 — відносне посилання (змінюється відносно нової позиції)
$A1 або A$1 — мішане посилання

🧩 Перевір себе: Абсолютні vs Відносні посилання

Що станеться з формулою =$A$1+B1 при копіюванні вниз на 1 рядок?

A стане =$A$2+B2
B стане =$A$1+B2
C стане =A1+$B$2

🎯 Логічні функції: IF, AND, OR, NOT

Навчимо таблицю приймати рішення!

Функція Синтаксис Що робить? Приклад із життя
IF (ЯКЩО) =IF(умова; так; ні) Перевіряє умову і повертає одне з двох значень Якщо бал ≥ 9 → "Відмінно", інакше → "Добре"
AND (І) =AND(умова1; умова2) Повертає ІСТИНА, якщо ВСІ умови виконані Знижка, якщо покупка > 1000 грн І клієнт VIP
OR (АБО) =OR(умова1; умова2) Повертає ІСТИНА, якщо ХОЧА Б одна умова виконана Вихідний, якщо субота АБО неділя
NOT (НЕ) =NOT(умова) Інвертує результат (істина → хибність) НЕ дорослий → дитина

🛠️ Практична робота

Завдання: Створи таблицю розрахунку зарплати з преміями

⭐ IF для перевірки плану ⭐ AND для премії ⭐ OR для надбавок

📋 Інструкція:

  1. Відкрий Google Таблицю за посиланням нижче
  2. Створи стовпці: Працівник, Продажі, План виконано, Премія
  3. Використай =IF(B2>=100;"Так";"Ні") для перевірки плану
  4. Додай =IF(AND(B2>=100;C2="Так");B2*0.1;0) для розрахунку премії
